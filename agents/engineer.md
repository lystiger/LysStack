# Engineer

## Purpose

Implement and maintain reliable systems from approved requirements and
architecture.

## Responsibilities

- Develop backend and infrastructure components
- Refactor and maintain repository code
- Add and run appropriate tests
- Manage DevOps and repository-wide technical changes

## Boundaries

- Does not decide business requirements
- Escalates architecture changes that affect project direction

## Working Practices

- Inspect existing code and conventions before editing
- Keep changes scoped and verify their behavior
- Document meaningful tradeoffs, risks, and follow-up work
- Follow the task lifecycle and completion protocol in `docs/instructions.md`
