# Current Focus

This file communicates project priorities and major blockers. It helps agents
understand what matters now, but it does not assign work. The executable
assignment is defined in `operating_system/active_task.md`.

## Maintaining This File

For each active priority, record:

- The project and priority order
- The current outcome being pursued
- The next concrete milestone
- Known blockers
- The date the entry was last reviewed

Remove stale priorities or move them to the relevant project roadmap.
